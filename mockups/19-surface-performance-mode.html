Surface View: Performance / Project Mode

Performance Mode

Cross-track performance interface. The AI organizes the project's tracks into musical sections and provides big, gestural controls for live performance and composition. Think DJ mixer meets modular performance panel.

Global Gestures

  • Energy -- horizontal fader controlling density, velocity, and filter brightness across all tracks. Left = stripped back, right = full energy.
  • Tension -- horizontal fader controlling build/release. Right = building (fills, risers, activity). Left = release (sparse, resolved).
  • Scene -- crossfader blending all mapped parameters between two scene snapshots (e.g. Verse / Chorus). The AI or human names the scenes.

Section Groups

The AI groups tracks by musical role. Each section gets its own card with a distinct accent color. Sections for this project:

  • Drums (Kick, Snare, HiHat) -- rose accent
  • Bass (Bass) -- indigo accent
  • Melodic (Lead, Pad) -- amber accent

Other possible sections: FX, Vocals, Pads, Percussion. The AI decides groupings based on track roles.

Per-Section Controls

  • Kill switch -- big 60x60 button. Mutes/unmutes the entire section. Lit when active, dim when killed.
  • Filter -- large 64px knob. Center = flat. Left = low pass closing. Right = high pass opening. Sweeps across all tracks in the section.
  • Level -- large 64px knob. Group volume for the section.
  • Character macros -- optional 52px knobs the AI adds for sections with expressive parameters (e.g. "Character" on bass, "Space" on melodics).
  • Level meter -- combined output of the section.

Principle

Big gestures, not individual parameters. DJ-style. Controls are large and spaced out, with generous negative space. This is a performance interface, not a configuration panel.

Mode Toggle

Track shows per-track Surface modules (the AI-curated controls for a single selected track). Project shows this cross-track performance view with section groups and global gestures.
