The Coin
A persistent round button in the bottom-right corner on every tab. It toggles between Chat and the last instrument tab. On instrument tabs, a notification card can appear attached to the coin when Gluon has something worth showing.

Reference
The Coin
  • Always present on every tab, same position (bottom-right)
  • 36px round button, var(--bg-raised) background
  • Contains an 8px status dot centered inside
  • Click or Cmd+K toggles between Chat and last instrument tab
  • Identical appearance on Chat and instrument tabs when idle
Notification Card
  • Appears on instrument tabs only, attached to the coin's left side
  • Card + coin form a single visual unit (shared background, continuous shape)
  • Transient -- collapses back to just the coin when there is nothing to show
  • Max width ~280px, DM Mono 11px, compact padding (8px 12px)
  • Auto-dismisses after task completion
States
  • Idle: coin only, dot in var(--text-faint)
  • Working: coin + minimal card, dot pulses in emerald
  • Needs attention: coin + question card, dot in amber
  • Complete: coin + result card, dot solid emerald, auto-dismiss
Keyboard
  • Cmd+K toggles between Chat and last instrument tab
  • On Chat: Cmd+K returns to last instrument tab
  • On instrument tabs: Cmd+K opens Chat
  • Remembers which instrument tab you came from
Design intent: The coin is the anchor, the card is transient intelligence around it. The coin never moves, never changes shape, and is always available for the toggle. The notification card grows outward from it only when Gluon has something worth showing -- a question, a progress indicator, a completion message. When there is nothing to say, it is just a small round button in the corner that takes up almost no space.
